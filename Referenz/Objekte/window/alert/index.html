Die Methode alert() gibt ein Dialogfenster aus. Sie besitzt einen möglichen Parameter der einen Text, eine Zahl oder ein Objekt sein kann.

Beispiel 1 - Text ausgeben

{% highlight javascript %} alert("Hallo Welt"); {% endhighlight %}

Beispiel 2 - Zahl ausgeben

{% highlight javascript %} alert(123); {% endhighlight %}

Neben festen Zeichenketten und Zahlen können Sie der alert()-Methode auch Variablen übergeben. In einem Solchen Fall wird der Wert der Variablen ausgegeben.

Beispiel 3 - Variable übergeben

{% highlight javascript %} var zeichenkette = "Hallo Welt"; alert(zeichenkette); {% endhighlight %}

Hinweise

  • Die alert()-Methode wandelt intern den übergebenen Wert immer in eine Zeichenkette um.
  • Der Rückgabewert ist immer undefined.
  • Das Dialogfenster kann je nach Betriebssystem und Browser unterschiedlich aussehen.
