Tipps & Tricks

Tabs innerhalb von pre-Tags ersetzen

Das folgende Beispiel zeigt, wie man Tabs innerhalb von pre-Tags durch geschützte Leerzeichen (&nbsp;) ersetzen kann. Als Hilfe wird dazu jQuery verwendet.

{% highlight javascript %} $('pre').html(function() { return this.innerHTML.replace(/\t/g, '  '); }); {% endhighlight %}

Diese Änderung sollte man aber immer erst durchführen lassen, wenn das Document Object Model (DOM) bereit ist für die Ausführung von JavaScript. Wie man dies realisieren kann, zeigt das folgende Beispiel.

{% highlight javascript %} $(function() { $('pre').html(function() { return this.innerHTML.replace(/\t/g, '  '); }); }); {% endhighlight %}
