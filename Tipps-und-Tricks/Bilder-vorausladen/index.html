Bilder vorausladen

Das folgende Beispiel zeigt, wie man Bilder vorausladen (preloading) kann.

{% highlight javascript %} img1 = new Image(); img1.src = 'images/image1.png'; img2 = new Image(); img2.src = 'images/image2.png'; {% endhighlight %}
