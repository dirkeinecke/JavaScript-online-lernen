Kilometer in Meilen umrechnen

Das folgende Beispiel zeigt, wie man mittels JavaScript Kilometer in Meilen umrechnen kann.

{% highlight javascript %} function convert_km_to_miles(km, decimal_place) { miles = parseInt(km) * 0.621371; if (decimal_place != -1) { var a = Math.pow(10, decimal_place); miles = Math.round(miles * a) / a; } return miles; } miles = convert_km_to_miles(100, 2); {% endhighlight %}
