Erste Schritte in JavaScript

JavaScript-Bibliotheksdateien

Bisher haben wir den JavaScript-Code immer direkt in die HTML-Datei geschrieben. Hier noch einmal zur Erinnerung unser erstes "Hallo-Welt"-Beispiel:

{% highlight html %} JavaScript {% endhighlight %}

Wenn Ihr JavaScript-Code aber in Zukunft immer umfangreicher wird, dann kann es mit dieser Methode recht sehr unübersichtlich werden. Auch wenn Sie wiederverwendbare Funktionen (ich werde später genauer auf das Thema Funktionen eingehen) verwenden möchten, dann müssten Sie mit der bisherigen Methode diese Funktionen immer manuell von einer zur nächsten Datei kopieren. Das wäre sehr mühsam und zudem auch noch fehleranfällig.

JavaScript bietet Ihnen hier die Möglichkeit, den Code in separate Bibliotheksdateien auszulagern. Bei diesen Bibliotheksdateien handelt es sich um einfache Textdateien mit der Dateiendung js.
