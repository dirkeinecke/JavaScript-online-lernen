Erste Schritte in JavaScript

Kein JavaScript im Browser verfügbar

Alle modernen Browser können JavaScript ausführen. Aber ich schreibe an dieser Stelle ganz bewußt können. Fast alle Browser bieten ihren Benutzern die Möglichkeit, die Ausführung von JavaScript zu deaktivieren.

Für den Fall, dass der Browser kein JavaScript ausführen darf, gibt es das noscript-Tag. Mit diesem Tag kann man zum Beispiel einen Hinweis unterbringen, dass für die Webseite JavaScript aktiviert sein muss. Das folgende Beispiel zeigt dies.

Früher waren neben den Browsern, die JavaScript verstehen und interpretieren konnten, auch Browser die dazu nicht in der Lage waren an der Tagesordnung. Heutzutage ist dies (bis auf ganz wenige Ausnahmen) nicht mehr der Fall und man muss auf solche Browser keine spezielle Rücksicht mehr nehmen. Ich möchte aber an dieser Stelle trotzdem kurz beschreiben, was man damals gemacht hat. Denn ab und zu werden Sie im Internet sicher noch entsprechende Beispiele finden und ich möchte, dass Sie verstehen, was dort warum gemacht wurde.

Es gab also Browser, die mit JavaScript nichts anfangen konnten. Sie kannten die HTML-Tags script und noscript nicht. In HTML werden vom Browser unbekannte Tags einfach übergangen. Der Inhalt zwischen Start- und Endelement wird wie ganz normaler Text dargestellt. Und hier liegt das Problem. Nehmen wir zum Beispiel folgenden Quelltext:

{% highlight html %} {% endhighlight %}

Ein moderner Browser würde mit aktiviertem JavaScript drei Dialogboxen hintereinander anzeigen und der Text innerhalb des noscript-Tags wäre nicht nichtbar. Würde man JavaScript deaktiviere, würden der Inhalt des noscript-Tags angezeigt werden.

Wenn ein Browser mit JavaScript generell nichts anzufangen weiß, dann sieht die Ausgabe unseres Beispiels so aus:

{% highlight text %} alert("Hallo Welt!"); alert("Das ist JavaScript."); alert("Das ist eine Dialogbox."); Für diese Webseite muss JavaScript aktiviert sein. {% endhighlight %}

Die Tags script und noscript sind unbekannt und werden einfach weggelassen. Der Inhalt wird wie ganz normaler Text behandelt.
