Erste Schritte in JavaScript

In diesem Kapitel lernen Sie die grundlegenden Schritte für das Erstellen und Ausführen von JavaScript-Programmen kennen. Besonderes Augenmerkt wird dabei auf das Einbinden von JavaScript in eine HTML-Datei gelegt. Diese HTML-Datei wird dann im Browser geöffnet und der JavaScript-Code wird durch den Browser ausgeführt.

HTML-Datei mit JavaScript-Code

In diesem ersten Beispiel wird gezeigt, wie man den Text "Hallo Welt" mittels JavaScript im Browser anzeigen kann.

Im zweiten Schritt erweitern wir nun dieses HTML-Grundgerüst mit dem JavaScript-Code, der im Browser ein Dialogfenster mit dem Text "Hallo Welt" anzeigt. Dies ist zwar nicht sonderlich spektakulär, zeigt Ihnen aber sehr anschaulich die Grundprinzipien.

Speichern Sie die Datei "Hallo-Welt.html" nach Ihren Änderungen wieder und wechseln Sie zum Browser. Hier sollte noch die Datei "Hallo-Welt.html" geöffnet sein. Aktualisieren Sie die Ansicht (Beispiel). Sie sehen nun ein Dialogfenster mit dem Text "Hallo Welt". Je nach Browser kann dieses Fenster immer etwas anders aussehen.

Nachdem wir nun unser erstes Erfolgserlebnis hatten, wollen wir uns gemeinsam den neu hinzugekommenen Quellcode etwas genauer ansehen.

Zunächst ist das script-Tag hinzugekommen. Dieses Tag sagt dem Browser, dass nun der Quellcode einer Scriptsprache folgt. Standardmäßig nehmen die meisten Browser an dieser Stelle an, dass es sich um JavaScript handelt. Sicherheitshalber geben wir über das type-Attribut beim Startelement die Sprache mit dem Wert text/javascript explizit an, da man nicht immer davon ausgehen kann, dass JavaScript als Standard-Scriptsprache für das script-Tag gesetzt ist.

Hinweis

Früher wurde die Scriptsprache über das Attribut language angegeben:

Dies ist jedoch falsch. Durch den HTML-Standard wird die Angabe des type-Attributs vorgegeben und muss von Ihnen auch genau so verwendet werden.

Sie können beliebig viele script-Bereiche in Ihre HTML-Datei einbauen und so beliebig zwischen HTML-Code und JavaScript-Code wechseln.

Zwischen dem Start- und dem Endelement des script-Tags steht der eigentliche JavaScript-Code, der in unserem Beispiel für die Ausgabe des Dialogfensters verantwortlich ist.

{% highlight javascript %} alert("Hallo Welt"); {% endhighlight %}

Mit dieser Zeile JavaScript-Code sagen Sie dem Computer, dass er ein Dialogfenster öffnen soll (alert()) und darin den Text "Hallo Welt" ausgeben soll. alert() ist dabei der Methodenaufruf für die Anzeige eines Dialogfensters. Der Text, der in diesem Dialogfenster angezeigt werden soll, wird als Parameter übergeben. Die Anführungszeichen bei diesem Parameter zeigen in unserem Fall, dass es sich um eine Zeichenkette handelt und sie werden nicht mit ausgegeben. Sie können hier sowohl die doppelten Anführungszeichen ("Hallo Welt") oder einfache Anführungszeichen ('Hallo Welt') verwenden.

Interaktion mit dem Benutzer

In den bisherigen Beispielen wurde der JavaScript-Code beim Laden der Webseite gleich ausgeführt. Manchmal kann dies so gewollt sein. Oft ist es allerdings so, dass man auf eine Aktion des Benutzers reagieren möchte. Es soll also eine Interaktion zwischen dem Benutzer und der Webseite stattfinden.

In der Programmiersprache spricht man an dieser Stelle von Ereignissen (Events), die durch den Benutzer ausgelöst werden. Wir werden hier am Anfang unserer Reise durch die JavaScript-Welt zunächst nur die Grundprinzipien dieser "Ereignisse" betrachten und dann später tiefer in dieses Thema einsteigen. Warum? "Ereignisse" machen die Arbeit mit JavaScript spanender, doch die kompette Darstellung dieses komplexen Bereiches würde hier am Anfang zu weit führen.

Kommentare

Genauso wie in HTML und anderen Programmiersprachen gibt es auch in JavaScript Kommentare. Diese dienen dazu, die Lesbarkeit Ihres Quelltextes zu verbessern. Haben Sie zum Beispiel eine Datei mit mehreren hundert Zeilen Code, dann ist es für Sie und andere Programmierer sehr hilfreich, wenn an einigen Stellen Hinweise stehen. Vom ausführenden Computer werden kommentare nicht beachtet. Er ignoriert sie einfach.

JavaScript kennt einzeilige und mehrzeilige Kommentare. Schauen wir uns zunächst die einzeiligen Kommentare an.
